# Security Policy

## Supported Versions

- Beta versions are released at 2.0.xbt

- Stable versions are released at 2.0.x

### Python

| Stable | Supported          |
| ------ | ------------------ |
| 2.0.0  | :white_check_mark: |

### Typescript

| Stable | Supported          |
| ------ | ------------------ |
| 2.0.0  | :white_check_mark: |

## Reporting a Vulnerability

- If you notice a vulnerability in our project, please report it privately using GitHub's security advisory reporting feature for this repository.

- Or if you have already resolved the issue, you can submit a pull request (https://github.com/Ahen-Studio/mesosphere-backend/pulls).

We would be delighted to help you make our project more secure.
