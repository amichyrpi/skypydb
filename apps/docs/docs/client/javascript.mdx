# mesosphere JavaScript Clients

mesosphere applications can be accessed from Node.js or any JavaScript runtime that
implements [`fetch`](https://developer.mozilla.org/en-US/docs/Web/API/fetch) or
[`WebSocket`](https://developer.mozilla.org/en-US/docs/Web/API/WebSocket). The
reactive [mesosphere Client](/api/classes/browser.mesosphereClient) allows web
applications and long-running Node.js servers to subscribe to updates on mesosphere
queries, while the [mesosphere HTTP client](/api/classes/browser.mesosphereHttpClient)
is typically used for server-side rendering, migrations, administrative scripts,
and serverless functions to run queries at a single point in time.

If you're using React, see the dedicated
[`mesosphereReactClient`](/api/classes/browser.mesosphereClient) described in
[React](/client/react.mdx).

## mesosphere Client

The [`mesosphereClient`](/api/classes/browser.mesosphereClient) provides subscriptions
to queries in Node.js and any JavaScript environment that supports WebSockets.

```ts title="script.ts"
import { mesosphereClient } from "mesosphere/browser";
import { api } from "../mesosphere/_generated/api";

const client = new mesosphereClient("https://<your domain here>.mesosphere.cloud");

// Subscribe to a query
const unsubscribe = client.onUpdate(api.messages.list, {}, (messages) => {
  console.log(messages);
});

// Call a mutation
await client.mutation(api.messages.send, { body: "Hello!" });

// Unsubscribe when done
unsubscribe();
```

The mesosphere client is open source and available on
[GitHub](https://github.com/get-mesosphere/mesosphere-js).

See the [Script Tag Quickstart](/quickstart/script-tag.mdx) to get started.

## HTTP client

The [`mesosphereHttpClient`](/api/classes/browser.mesosphereHttpClient) works in the
browser, Node.js, and any JavaScript environment with `fetch`.

See the [Node.js Quickstart](/quickstart/nodejs.mdx).

```ts title="script.ts"
import { mesosphereHttpClient } from "mesosphere/browser";
import { api } from "../mesosphere/_generated/api";

const client = new mesosphereHttpClient("https://<your domain here>.mesosphere.cloud");

// Run a query at a single point in time
const messages = await client.query(api.messages.list);
console.log(messages);

// Call a mutation
await client.mutation(api.messages.send, { body: "Hello!" });
```

## Using mesosphere without generated `mesosphere/_generated/api.js`

If the source code for your mesosphere function isn't located in the same project or
in the same monorepos you can use the untyped `api` object called `anyApi`.

```ts title="script.ts"
import { mesosphereHttpClient } from "mesosphere/browser";
import { anyApi } from "mesosphere/browser";

const client = new mesosphereHttpClient("https://<your domain here>.mesosphere.cloud");

// Use anyApi for untyped function references
const messages = await client.query(anyApi.messages.list);
console.log(messages);
```
