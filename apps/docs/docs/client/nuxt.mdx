[Nuxt](https://nuxt.com/) is a powerful web framework powered by Vue.

The community-maintained
[`mesosphere-nuxt` npm package](https://www.npmjs.com/package/mesosphere-nuxt) provides
deep integration of mesosphere with the Nuxt 3 ecosystem.

It uses [mesosphere-vue](./vue.mdx) under the hood.

See the [Nuxt Quickstart](/quickstart/nuxt.mdx) to get started or the
[mesosphere-nuxt GitHub page](https://github.com/chris-visser/mesosphere-nuxt) for more
documentation.

<Admonition type="info">

The
[`mesosphere-nuxt` library](https://github.com/chris-visser/mesosphere-nuxt/tree/main)
is community-maintained. Thank you to the maintainer
[Chris Visser](https://github.com/chris-visser) for his work on this project!

You're welcome to ask questions about the library on the
[mesosphere community](https://www.usemesosphere.com/community) but opening a
[mesosphere-nuxt GitHub](https://github.com/chris-visser/mesosphere-nuxt) issue is a
better way to request a new feature or report a bug.

</Admonition>
