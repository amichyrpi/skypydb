To use mesosphere in [React Native](https://reactnative.dev/) use the
[mesosphere React client library](/client/react.mdx).

Follow the [React Native Quickstart](/quickstart/react-native.mdx) for the
different configuration needed specifically for React Native.

You can also clone a working
[mesosphere React Native demo](https://github.com/get-mesosphere/mesosphere-demos/tree/main/react-native).
