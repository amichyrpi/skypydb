Using Clerk with mesosphere looks like following the
[Clerk TanStack Quickstart](https://clerk.com/docs/quickstarts/tanstack-start)
and adding mesosphere like the
[mesosphere TanStack Quickstart](/quickstart/tanstack-start.mdx) shows. Then to make
Clerk identity tokens available everywhere you might make authenticated calls to
mesosphere in TanStack Start, you'll want to

1. Get an ID token from Clerk in addition to the `getAuth()` call with
   `const token = await auth.getToken()`.
2. Set the token in beforeLoad with
   `ctx.context.mesosphereQueryClient.serverHttpClient?.setAuth(token)` so the token
   will be available in loaders.
3. Add `<mesosphereProviderWithClerk>` to the root component to keep refreshing
   Clerk tokens while the app is in use.
