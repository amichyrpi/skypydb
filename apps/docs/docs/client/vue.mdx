[Vue](https://vuejs.org/) is an "approachable, performant and versatile
framework for building web user interfaces."

The community-maintained
[`mesosphere-vue`](https://www.npmjs.com/package/mesosphere-vue) npm package is a Vue.js
integration library for mesosphere. It also powers [mesosphere-nuxt](./nuxt.mdx).

See the [Vue Quickstart](/quickstart/vue.mdx) to get started or the
[mesosphere-vue GitHub page](https://github.com/chris-visser/mesosphere-vue) for more
documentation.

<Admonition type="info">

The [`mesosphere-vue` library](https://github.com/chris-visser/mesosphere-vue) is
community-maintained. Thank you to the maintainer
[Chris Visser](https://github.com/chris-visser) for his work on this project!

You're welcome to ask questions about the library on the
[mesosphere community](https://www.usemesosphere.com/community) but opening a
[mesosphere-vue GitHub](https://github.com/chris-visser/mesosphere-vue) issue is a
better way to request a new feature or report a bug.

</Admonition>
