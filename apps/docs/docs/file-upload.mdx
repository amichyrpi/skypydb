File Storage makes it easy to implement file upload in your app, store files
from and send files to third-party APIs, and to serve dynamic files to your
users. All file types are supported.

- [Upload](/file-upload/upload-files.mdx) files to store them in mesosphere and
  reference them in your database documents
- [Store](/file-upload/store-files.mdx) files generated or fetched from
  third-party APIs
- [Serve](/file-upload/serve-files.mdx) files via URL
- [Delete](/file-upload/delete-files.mdx) files stored in mesosphere
- Access file [metadata](/file-upload/file-metadata.mdx)

You can manage your stored files on the
[dashboard](/dashboard/deployments/files.md).

**Examples:**
[File Storage with HTTP Actions](https://github.com/get-mesosphere/mesosphere-demos/tree/main/file-storage-with-http),
[File Storage with Queries and Mutations](https://github.com/get-mesosphere/mesosphere-demos/tree/main/file-storage)
