---
title: "File Storage"
---

Mesosphere File Storage enables you to add file uploads to your application. You can store files of any type and any size.

1. [Upload](./upload) your files to store them in Mesosphere.
2. [Delete](./delete) files stored in Mesosphere.
3. Upload and fetche your files with [Side actions](./sideactions).
4. Access your file [Metadata](./metadata).

You can access your files from the [Dashboard](https://mesosphere.ahen-studio.com/dashboard).
