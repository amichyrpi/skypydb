title: Bun
description: "Add Mesosphere to a Bun project"
---

Learn how to use read and write functions in your React app.

1. **Create a new Bun project.**

```shell shell
bun init -y
```

2. **After creating your project**, install the Mesosphere package:

```shell shell
bun add mesosphere
```

3. **Start the Mesosphere dev server** in a separate terminal window, it will automatically create a `mesosphere/` folder containing a `deploy.ts` file and a `.env` file in your project's root directory, don't forget to keep the command (`bunx mesosphere dev`) running in the background. It will automatically update your functions when you make changes:

```shell shell
bunx mesosphere dev
```

4. **Create a write and read functions** in the `mesosphere/` folder. For example, create a file called `tasks.ts` and add the following code, it will export `api.tasks.newtask` and `api.tasks.readtask`:

```typescript mesosphere/tasks.ts
import { readFunction, writeFunction } from "mesosphere/reactlibrarie";
import { type } from "mesosphere/type";

// Add a task to the database with a task name and a boolean to indicate if it succeed or not.
export const newtask = writeFunction({
  args: {
    task: type.string(),
    succeed: type.boolean(),
  },
  handler: async (mesosphere, args) => {
    await mesosphere.database.add("tasks", {
      task: args.task,
      succeed: args.succeed,
    });
  },
});

// Read all tasks from the database.
export const readtask = readFunction({
  args: {},
  handler: async (database) => {
    return await database.get("tasks");
  },
});
```

**What it does:** It creates a new task in the database with a task name and a boolean to indicate if it succeed or not, and it exports a read function that returns all tasks from the database.

5. **Connect your bun application to your backend.** Create a new file name `app.ts`, create a `MesosphereHttpClient` using the url and the api key of your development environment:

```typescript app.ts
import { MesosphereHttpClient } from "mesosphere/http";
import { api } from "../mesosphere/deploy";

const client = new MesosphereHttpClient(
  process.env[("MESOSPHERE_URL", "MESOSPHERE_API_KEY")],
);

client.write(api.tasks.newtask, {}, { task: "task1", succeed: True });

// `hug` the data feed to always stay up to date.
const hugger = client.hug(api.tasks.newtask, {}, async (tasks) => {
  console.log(tasks);
});

await Bun.sleep(500);
hugger();
await client.close();
```

7. **Run your app.** You can now run your app with `bun app.ts` and see the tasks in your app.

See the full [Bun documentation](/docs/client/javascript/bun).
