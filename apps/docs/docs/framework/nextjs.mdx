<Tip>

Next.js + Mesosphere auth

Mesosphere integrates quickly and easily with Next.js to add [auth](/docs/client/nextjs/approuter) to your web app.

</Tip>

To get up and running with Mesosphere in a Next.js project, run the following command:

```shell shell
npm create mesosphere@latest
```

or follow the guide below.

Learn how to use read and write functions in your Next.js app.

1. **Create a Next.js app** using create next command.

```shell shell
npx create-next-app@latest my-functions
```

2. **After creating your app**, install the Mesosphere package, but first you need to `cd` into the project:

```shell shell
cd my-functions
npm install mesosphere
```

3. **Start the Mesosphere dev server** in a separate terminal window, it will automatically create a `mesosphere/` folder containing a `deploy.ts` file and a `.env` file in your project's root directory, don't forget to keep the command (`npx mesosphere dev`) running in the background. It will automatically update your functions when you make changes:

```shell shell
npx mesosphere dev
```

4. **Create a write and read functions** in the `mesosphere/` folder. For example, create a file called `tasks.ts` and add the following code, it will export `api.tasks.newtask` and `api.tasks.readtask`:

```typescript mesosphere/tasks.ts
import { readFunction, writeFunction } from "mesosphere/reactlibrarie";
import { type } from "mesosphere/type";

// Add a task to the database with a task name and a boolean to indicate if it succeed or not.
export const newtask = writeFunction({
  args: {
    task: type.string(),
    succeed: type.boolean(),
  },
  handler: async (mesosphere, args) => {
    await mesosphere.database.add("tasks", {
      task: args.task,
      succeed: args.succeed,
    });
  },
});

// Read all tasks from the database.
export const readtask = readFunction({
  args: {},
  handler: async (database) => {
    return await database.get("tasks");
  },
});
```

**What it does:** It creates a new task in the database with a task name and a boolean to indicate if it succeed or not, and it exports a read function that returns all tasks from the database.

5. **Create a Mesosphere Client Provider component.** Passe the `ReactClient` component to `MesosphereProvider`, create a new file in the `app/` folder name `MesosphereClientProvider.tsx` with the following code:

```typescript app/MesosphereClientProvider.tsx
"use client";

import { MesosphereProvider, ReactClient } from "mesosphere/reactlibrarie";
import { ReactNode } from "react";

// Initialize the Mesosphere client with the URL and API key.
const mesosphere = new ReactClient(
  import.meta.env.VITE_MESOSPHERE_URL,
  import.meta.env.VITE_MESOSPHERE_API_KEY,
);

export function MesosphereClientProvider({ children }: { children: ReactNode }) {
  return <MesosphereProvider client={mesosphere}>{children}</MesosphereProvider>;
}
```

6. **Connect MesosphereClientProvider to your Next.js app.** In the `app/layout.tsx` file, wrap your app with `MesosphereClientProvider` it:

```typescript app/layout.tsx
import type { Metadata } from "next";
import localFont from "next/font/local";
import "./globals.css";
import { MesosphereClientProvider } from "./MesosphereClientProvider";

const geistSans = localFont({
  src: "./fonts/GeistVF.woff",
  variable: "--font-geist-sans",
});
const geistMono = localFont({
  src: "./fonts/GeistMonoVF.woff",
  variable: "--font-geist-mono",
});

export const metadata: Metadata = {
  title: "Create Next App",
  description: "Generated by create next app",
};

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  return (
    <html lang="en">
      <body
        className={`${geistSans.variable} ${geistMono.variable} antialiased`}
      >
        <MesosphereClientProvider>{children}</MesosphereClientProvider>
      </body>
    </html>
  );
}
```

7. **Use the functions in your Next.js app.** in the `app/page.tsx` file, you can use the `newtask` and `readtask` functions to add and read tasks from the database:

```typescript app/page.tsx
"use client";

import { callread, callwrite } from "mesosphere/reactlibrarie";
import { api } from "../mesosphere/deploy";

export default function Home() {
  // Read all the tasks from the database.
  const readtask = callread(api.tasks.readtask);
  // Write a new message to the database.
  const newtasks = callwrite(api.tasks.newtask);

  // Add a task to the database with a task name and a boolean to indicate if it succeed or not.
  const addtack = await newtasks({ task: This is a task, succeed: True });

  // Read all tasks from the database.
  const tasks = await readtask({});

  return (
    <main className="flex min-h-screen flex-col items-center justify-between p-24">
      <button
        onClick={() => addtack()}>Add task
      </button>;
      {tasks?.map(({ _id, text }) => <div key={_id}>{text}</div>)}
    </main>
  );
}
```

8. **Run your app.** You can now run your app with `npm run dev` and see the tasks in your app.
