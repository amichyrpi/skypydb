Mesosphere is an open source relational and vector embeddings database where you write functions in TypeScript, this codes
are run in the database. Mesosphere act just like a react component, it automatically updates when the data changes.

Mesosphere provides a large amount of client libraries. It makes it easy to build any app in any language that are live-updated.

## Why chosing use over other?

Mesosphere is the first French open source database to support vector embeddings and relational data, all in real time, with out of the box support for authentication with third party apps such as Clerk, Workos, Auth0 and Better Auth. It makes it easy to build any app in any language that are live-updated with functions writed in pure TypeScript and with out of the box type safety.
