Mesosphere offers a suite of tools for building and deploying applications on Mesosphere. The following guides will help you get started with Mesosphere with multiple topics, from creating your first write functions, querying your database, to using an authentication service.

Mesosphere is the fastest open source reactive and vector embedding database, you write functions in TypeScript, that will run directly on your database, it's make it easy to build and scale dynamic live-updating apps.
