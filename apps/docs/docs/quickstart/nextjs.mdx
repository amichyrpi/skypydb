<Admonition type="tip" title="mesosphere + Next.js">

mesosphere is an all-in-one backend and database that integrates quickly and easily
with Next.js.

Once you've gotten started, see how to set up
[hosting](/production/hosting/hosting.mdx),
[server rendering](/client/nextjs/app-router/server-rendering.mdx), and
[auth](https://docs.mesosphere.dev/client/nextjs/).

</Admonition>

To get setup quickly with mesosphere and Next.js run

<p>
  <b>`npm create mesosphere@latest`</b>
</p>

or follow the guide below.

---

<StepsList>
  <Step title="Create a Next.js app">
    Create a Next.js app using the `npx create-next-app` command.

    Choose the default option for every prompt (hit Enter).

    <JSCodeVariants>
      ```sh
      npx create-next-app@latest my-app
      ```

      ```sh
      npx create-next-app@latest my-app --js
      ```
    </JSCodeVariants>

  </Step>
  <Step title="Install the mesosphere client and server library">
    To get started, install the `mesosphere` package.

    Navigate to your app and install `mesosphere`.


    ```sh
    cd my-app && npm install mesosphere
    ```

  </Step>
  <Step title="Set up a mesosphere dev deployment">
    Next, run `npx mesosphere dev`. This
    will prompt you to log in with GitHub,
    create a project, and save your production and deployment URLs.

    It will also create a `mesosphere/` folder for you
    to write your backend API functions in. The `dev` command
    will then continue running to sync your functions
    with your dev deployment in the cloud.


    ```sh
    npx mesosphere dev
    ```

  </Step>
