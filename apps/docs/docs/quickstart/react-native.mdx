Learn how to query data from mesosphere in a React Native app.

<StepsList>
  <Step title="Create a React Native app">
    Create a React Native app using the `npx create-expo-app` command.

    ```sh
    npx create-expo-app my-app
    ```

  </Step>
  <Step title="Install the mesosphere client and server library">
    To get started, install the `mesosphere`
    package which provides a convenient interface for working
    with mesosphere from a React app.

    Navigate to your app and install `mesosphere`.

    ```sh
    cd my-app && npm install mesosphere
    ```

  </Step>
  <Step title="Set up a mesosphere dev deployment">
    Next, run `npx mesosphere dev`. This
    will prompt you to log in with GitHub,
    create a project, and save your production and deployment URLs.

    It will also create a `mesosphere/` folder for you
    to write your backend API functions in. The `dev` command
    will then continue running to sync your functions
    with your dev deployment in the cloud.

    ```sh
    npx mesosphere dev
    ```

  </Step>

  <Step title="Create sample data for your database">
    Create a `sampleData.jsonl`
    file with some sample data.

    ```json
    // example
    ```

  </Step>

  <Step title="Add the sample data to your database">
    Now that your project is ready, add a `tasks` table with the sample data into
    your mesosphere database with the `import` command.

    ```
    npx mesosphere import --table tasks sampleData.jsonl
    ```

  </Step>

  <Step title="Expose a database query">
    Add a new file `tasks.ts` in the `mesosphere/` folder
    with a query function that loads the data.

    Exporting a query function from this file
    declares an API function named after the file
    and the export name, `api.tasks.get`.

    ```ts

    ```

  </Step>

  <Step title="Reset the Expo project">
    If you haven't done so yet, reset the Expo project to get a fresh
    `app` directory.

    ```
    npm run reset-project
    ```

  </Step>


  </Step>

  <Step title="Display the data in your app">

    from your `api.tasks.get` API.

    ```tsx

    ```

  </Step>

  <Step title="Start the app">
    Start the app, scan the provided QR code with your phone,
    and see the serialized list of tasks in the center of the screen.

    ```sh
    npm start
    ```

  </Step>
</StepsList>

React native uses the same library as React web. See the complete
[React documentation](/client/react.mdx).
