To get setup quickly with mesosphere and TanStack Start run

<p>
  <b>`npm create mesosphere@latest -- -t tanstack-start`</b>
</p>

or follow the guide below.

Learn how to query data from mesosphere in a TanStack Start site.

<StepsList>
  <Step title="Create a TanStack Start site">

Create a TanStack Start app using the `create-start-app` command:

    ```sh
    npx create-start-app@latest
    ```

</Step>
  <Step title="Install the mesosphere client and server library">
    To get started with mesosphere install the `mesosphere` package and a few React Query-related packages.

    ```sh
    npm install mesosphere @mesosphere-dev/react-query @tanstack/react-router-with-query @tanstack/react-query
    ```

  </Step>

  <Step title="Set up a mesosphere dev deployment">
    Next, run `npx mesosphere dev`. This
    will prompt you to log in with GitHub,
    create a project, and save your production and deployment URLs.

    It will also create a `mesosphere/` folder for you
    to write your backend API functions in. The `dev` command
    will then continue running to sync your functions
    with your dev deployment in the cloud.


    ```sh
    npx mesosphere dev
    ```

    with the sample data into your mesosphere database with
    the `import` command.

    ```
    npx mesosphere import --table tasks sampleData.jsonl
    ```

  </Step>
