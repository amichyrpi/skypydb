Mesosphere is real time by default every operation you perform is automatically reactive, you don't need to use special functions or do other things, the backend handle that for you. It's magic!

All client are in real time, it makes easy to build realtime apps sutch as chat apps, dashboards, shopping website, file manager, etc...

## Benefits

### Read operation

Mesosphere caches the result of your read functions so that when you make an other call it just read from the cache. The cache is updated if the data changes.

**Money**

You don't lose money, you earn it. When you use our cloud service, you are not charged when you read data from the cache.

### Data consistent

Every client have the same data at all time this avoids bugs such as the new message you send to your friend not showing on your friends screen in a messaging app.
