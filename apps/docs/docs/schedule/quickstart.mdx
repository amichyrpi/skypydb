---
title: "Scheduling"
---

Mesosphere offers the ability to schedule your functions to run once or repeatedly in the future. This allows you to create workflows that run over time, such as sending verification emails after a certain delay or regularly reconciling your accounts with Stripe.

**There are two main ways to schedule functions in Mesosphere:**

1. [Scheduler](./scheduler) allows functions to be scheduled durably by other functions to run at a later point in time. You can schedule functions minutes, days, or even months in the future.

2. [Cron](./cron) schedules functions to run on a recurring basis, such as daily.
