# The secret behind our speed

At Mesosphere, we made a deliberate architectural choice for hosting our database. The computing layer runs entirely in random access memory (RAM), while persistent storage is handled on disk.

This approach allows us to deliver a blazing-fast backend with features that traditional backends often cannot provide.
