Mesosphere is an open-source relational and vector embeddings database. We are currently working on an early beta version **2.0.0-beta**.

# Contribute

You are welcome to contribute to the Mesosphere backend! We accept contributions of all kinds.

- Submit pull requests on [GitHub Pull Requests](https://github.com/Ahen-Studio/mesosphere-backend/pulls)
- Report issues on [GitHub Issues](https://github.com/Ahen-Studio/mesosphere-backend/issues)

# Contribution Process

We use a trust-based system for pull requests called [Vouch](https://github.com/mitchellh/vouch) by [Mitchell Hashimoto](https://github.com/mitchellh).

- PRs are flagged with a `vouch:*` label indicating the trust status of the contributor.
- External contributors are labeled `vouch:unvouched` until explicitly added to [.github/VOUCHED.td](https://github.com/Ahen-Studio/mesosphere-backend/blob/main/.github/VOUCHED.td).

For more details on contributing, see the [Contribution Guide](https://github.com/Ahen-Studio/mesosphere-backend/blob/main/CONTRIBUTING.md).
