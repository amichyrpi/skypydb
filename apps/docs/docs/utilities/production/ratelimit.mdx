title: "Rate Limiter"
---

Rate limits are calculated on the Mesosphere server whenever you access it.

You are billed per function call and per write operation, as well as for file bandwidth, database bandwidth, file uploads, vector bandwidth, and vector storage.

We currently offer two plans. A free plan is planned for a future public release.

> Due to European taxes, hosting your app in Europe costs approximately **30% more** than US hosting.
