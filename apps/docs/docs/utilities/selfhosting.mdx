Mesosphere is 100% open source that mean you can host it yourself. You can find more information on our [Github](https://github.com/Ahen-Studio/mesosphere-backend).

# Open Backend

The backend of Mesosphere is written in Rust, with the function environment written in TypeScript.

Mesosphere uses a [Apache 2.0](https://www.apache.org/licenses/LICENSE-2.0) license which allow you to do almost anything.

# Other product

At Ahen-Studio, we prioritize open source; that's why all products related to Mesosphere and others are available on our [GitHub](https://github.com/Ahen-Studio).
