Whether building RAG enabled chatbots or quick search in your applications,
mesosphere provides easy apis to create powerful AI and search enabled products.

[Vector Search](/vector/vector-search.mdx) enables searching for documents based
on their semantic meaning. It uses vector embeddings to calculate similarity and
retrieve documents that are similar to a given query. Vector search is a key
part of common AI techniques like RAG.

[Full Text Search](/vector/text-search.mdx) enables keyword and phrase search
within your documents. It supports prefix matching to enable typeahead search.
mesosphere full text search is also reactive and always up to date like all mesosphere
queries, making it easy to build reliable quick search boxes.

[mesosphere Actions](functions/actions) easily enable you to call AI apis, save data
to your database, and drive your user interface.
