Mesosphere supports all the following types of values:

<div className="mesosphere-full-width">

| Mesosphere Type | TS/JS Type                                                                                                           | Example Usage          | Types validator                            | Json format     | Notes                                                                                                                                |
| --------------- | -------------------------------------------------------------------------------------------------------------------- | ---------------------- | ------------------------------------------ | --------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| Id              | [Id](./documentsids) ([string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#string_type)) | `doc._id`              | `type.id("tableName")`                     | string          | Unique identifier referencing a document in a table. See [Document IDs](./documentsids).                                             |
| Null            | [null](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#null_type)                            | `null`                 | `type.null()`                              | null            | Represents the absence of a value. `undefined` is not a valid Mesosphere value — use `null` instead.                                 |
| Int64           | [bigint](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#bigint_type)                        | `3n`                   | `type.int64()`                             | string (base10) | Supports signed 64-bit integers between −2^63 and 2^63−1. Serialized as strings in JSON to preserve precision.                       |
| Float64         | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#number_type)                        | `1.2`                  | `type.number()`                            | number / string | IEEE-754 double-precision floating point. Special values like `NaN` or `Infinity` are serialized as strings.                         |
| Boolean         | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#boolean_type)                      | `true`                 | `type.boolean()`                           | bool            | Accepts `true` or `false`.                                                                                                           |
| String          | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#string_type)                        | `"abc"`                | `type.string()`                            | string          | UTF-8 encoded. Must be a valid Unicode sequence and stay under the 1MB size limit.                                                   |
| Bytes           | [ArrayBuffer](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/ArrayBuffer)          | `new ArrayBuffer(8)`   | `type.bytes()`                             | string (base64) | Raw binary data passed as an `ArrayBuffer`. Serialized as base64 in JSON. Must stay under the 1MB size limit.                        |
| Array           | [Array](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array)                      | `[1, 2.3, "abc"]`      | `type.array(values)`                       | array           | Ordered list of values. Can contain up to **8192 entries**.                                                                          |
| Object          | [Object](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#objects)                            | `{a: "abc"}`           | `type.object({property: value})`           | object          | Plain object with predefined properties. Supports up to 1024 fields. Field names must be non-empty and cannot start with `$` or `_`. |
| Record          | [Record](https://www.typescriptlang.org/docs/handbook/utility-types.html#recordkeys-type)                            | `{"a": "1", "b": "2"}` | `type.record(keys, values)`                | object          | Object with dynamic typed keys. Keys must be ASCII, non-empty, and cannot start with `$` or `_`.                                     |
| Any             | any                                                                                                                  | `"abc"` / `1` / `{}`   | `type.any()`                               | any JSON value  | Accepts any valid Mesosphere value. Use sparingly — it disables type validation entirely.                                            |
| Union           | any                                                                                                                  | `"abc"` / `42`         | `type.union(type.string(), type.number())` | value           | Value must match **one of the provided validators**.                                                                                 |
| Literal         | any                                                                                                                  | `"abc"`                | `type.literal("abc")`                      | value           | Accepts only the exact literal value specified.                                                                                      |
| Optional        | any                                                                                                                  | `undefined` / `"abc"`  | `type.optional(type.string())`             | value / missing | Field may be omitted or hold a valid value. Omitted fields are not stored in the document.                                           |

</div>
