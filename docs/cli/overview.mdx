## Requirements

- Python 3.11 or later

## Installation

the CLI is automatically installed with the base installation.

```bash
pip install skypydb
```

Verify installation with:

```bash
skypydb --version
```

## Commands

**[`skypydb dev`](/cli/dev)** - Create new projects with interactive setup and access the dashboard for your application

## Getting Help

View help for any command:

```bash
skypydb --help
```
