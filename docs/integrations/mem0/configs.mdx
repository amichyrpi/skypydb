title: configs component
---

Skypydb mem0 config is HTTP-only.

Required keys:

1. `api_url`
2. `api_key`
3. `collection_name` (optional, defaults to `"mem0"`)

```python
from typing import Any, Dict
from pydantic import BaseModel, ConfigDict, Field, model_validator


class SkypyDBConfig(BaseModel):
    collection_name: str = Field("mem0", description="Default name for the collection")
    api_url: str = Field(..., description="Skypydb backend base URL")
    api_key: str = Field(..., description="Skypydb API key")

    @model_validator(mode="before")
    @classmethod
    def validate_extra_fields(cls, values: Dict[str, Any]) -> Dict[str, Any]:
        allowed_fields = set(cls.model_fields.keys())
        extra_fields = set(values.keys()) - allowed_fields
        if extra_fields:
            raise ValueError(
                f"Extra fields not allowed: {', '.join(extra_fields)}. "
                f"Use only: {', '.join(allowed_fields)}"
            )
        return values

    model_config = ConfigDict(arbitrary_types_allowed=True)
```
