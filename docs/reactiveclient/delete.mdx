---
title: ReactiveClient Delete (Deprecated)
description: "ReactiveClient was removed in favor of HttpClient"
---

<Warning>`ReactiveClient` is no longer available.</Warning>

Use `HttpClient`:

```python
from mesosphere import HttpClient

client = HttpClient(api_url="http://localhost:8000", api_key="local-dev-key")
client.relational("users").delete(where={"email": {"$eq": "theo@example.com"}})
```
