---
title: ReactiveClient Schema (Deprecated)
description: "Schema DSL runtime is removed; use HTTP schema admin API"
---

<Warning>Local runtime schema APIs were removed from SDK exports.</Warning>

Use `HttpClient.schema`:

```python
from skypydb import HttpClient

client = HttpClient(api_url="http://localhost:8000", api_key="local-dev-key")
client.schema.apply(
    {
        "tables": {
            "users": {
                "fields": {
                    "name": {"type": "string"},
                    "email": {"type": "string"},
                }
            }
        }
    }
)
```
