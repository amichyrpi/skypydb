## Examples

Examples are stored in the TypeScript [Examples](https://github.com/Ahen-Studio/skypydb/demo/examples/TypeScript/cloud) folder.

## Cli

Use the CLI to connect to the cloud service, generate an api key, and create a database.

```bash
npx skypydbhttpclient dev
# npx ... --help
# npx ... --version
# are availables
```
