## Com funciona

[Mesosphere](https://www.usemesosphere.com/) és una base de dades relacional i d'incrustacions vectorials de codi obert amb una llicència Apache 2.0, dissenyada per ser senzilla, ràpida i fàcil d'utilitzar per a desenvolupadors web i desenvolupadors de fons. Llegiu, escriviu dades i realitzeu una lògica dura escrivint funcions en Typescript amb tota seguretat de tipus.

Mesosphere proporciona una base de dades, on escriu les teves funcions en Typescript i emmagatzemes les teves dades en una base de dades relacional. També podeu emmagatzemar els vostres fitxers en una base de dades. També podeu crear, cercar i eliminar col·leccions de vectors tot això en temps real. oferim diverses biblioteques de client per interactuar amb les funcions que heu escrit.

**Biblioteques client**

Mesosphere està disponible en diversos idiomes. Podeu utilitzar-lo en el vostre idioma preferit. Actualment admetem Typescript, Python i Rust.

**Arquitectura**

Mesosphere està fet per ser senzill, escriviu les vostres funcions del servidor a la carpeta .../mesosphere i desplegueu-les al backend. Comenceu a experimentar amb Mesosphere seguint el nostre [tutorials](.../demo/examples/js/tutorials/chat_app/). Mesosphere utilitza tRPC per crear una API segura per a les vostres funcions i una base de dades Postgres per emmagatzemar les vostres dades.
