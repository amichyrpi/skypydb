## como funciona

[Mesosphere](https://www.usemesosphere.com/) es una base de datos de incrustaciones vectoriales y relacionales de código abierto con licencia Apache 2.0. Está diseñada para ser simple, rápida y fácil de usar para desarrolladores web y desarrolladores backend. Lea, escriba datos y realice lógica estricta escribiendo funciones en Typecript con total seguridad de tipos.

Mesosphere proporciona una base de datos, donde escribe sus funciones en Typecript y almacena sus datos en una base de datos relacional. También puede almacenar sus archivos en una base de datos. También puedes crear, buscar y eliminar colecciones de vectores, todo eso en tiempo real. Proporcionamos múltiples bibliotecas de cliente para interactuar con las funciones que escribió.

**Bibliotecas de clientes**

Mesosphere está disponible en varios idiomas. Puedes usarlo en tu idioma favorito. Actualmente admitimos Typecript, Python y Rust.

**Arquitectura**

Mesosphere está diseñado para ser simple: escriba las funciones de su servidor en la carpeta .../mesosphere e impleméntelas en el backend. Comience a experimentar con Mesosphere siguiendo nuestro [tutorials](.../demo/examples/js/tutorials/chat_app/). Mesosphere usa tRPC para crear una API con seguridad de tipos para sus funciones y una base de datos Postgres para almacenar sus datos.
