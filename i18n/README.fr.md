## Comment ça marche

[Mesosphere](https://mesosphere.ahen-studio.com) est une base de données d'intégration relationnelle et vectorielle open source avec une licence Apache 2.0, elle est conçue pour être simple, rapide et facile à utiliser pour les développeurs Web et les développeurs back-end. Lisez, écrivez des données et exécutez une logique dure en écrivant des fonctions dans Typescript avec une sécurité de type totale.

Mesosphere fournit une base de données dans laquelle vous écrivez vos fonctions en Typescript et stockez vos données dans une base de données relationnelle. Vous pouvez également stocker vos fichiers dans une base de données. Vous pouvez également créer, rechercher et supprimer des collections de vecteurs, tout cela en temps réel. nous fournissons plusieurs bibliothèques clientes pour interagir avec les fonctions que vous avez écrites.

**Bibliothèques clientes**

La mésosphère est disponible en plusieurs langues. Vous pouvez l'utiliser dans votre langue préférée. Nous prenons actuellement en charge Typescript, Python et Rust.

**Architecture**

Mesosphere est conçu pour être simple, écrivez vos fonctions de serveur dans le dossier ./mesosphere et déployez-les sur le backend. Commencez à expérimenter la mésosphère en suivant notre [tutorials](./demo/examples/js/tutorials/chat_app/). Mesosphere utilise tRPC pour créer une API de type sécurisé pour vos fonctions et une base de données Postgres pour stocker vos données.

## Intégration

Nous fournissons une couche d'intégration avec [mem0](https://github.com/mem0ai/mem0) pour que vous puissiez ajouter de la mémoire à un LLM en utilisant notre [integration](./demo/examples/python/integration/)
