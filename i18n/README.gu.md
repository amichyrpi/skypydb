## તે કેવી રીતે કામ કરે છે

[Mesosphere](https://www.usemesosphere.com/) એ Apache 2.0 લાઇસન્સ ધરાવતું ઓપન-સોર્સ રિલેશનલ અને વેક્ટર એમ્બેડિંગ ડેટાબેઝ છે જે વેબ ડેવલપર્સ અને બેકએન્ડ ડેવલપર્સ માટે સરળ, ઝડપી અને ઉપયોગમાં સરળ બનવા માટે ડિઝાઇન કરવામાં આવ્યું છે. સંપૂર્ણ પ્રકારની સલામતી સાથે ટાઇપસ્ક્રીપ્ટમાં ફંક્શન લખીને ડેટા વાંચો, લખો અને સખત તર્ક કરો.

મેસોસ્ફિયર ડેટાબેઝ પૂરો પાડે છે, જ્યાં તમે તમારા કાર્યોને ટાઇપસ્ક્રીપ્ટમાં લખો છો અને તમારા ડેટાને રિલેશનલ ડેટાબેઝમાં સ્ટોર કરો છો. તમે તમારી ફાઇલોને ડેટાબેઝમાં પણ સ્ટોર કરી શકો છો. તમે રીઅલટાઇમમાં વેક્ટર સંગ્રહો બનાવી, શોધી અને કાઢી પણ શકો છો. તમે લખેલા કાર્યો સાથે ક્રિયાપ્રતિક્રિયા કરવા માટે અમે બહુવિધ ક્લાયન્ટ લાઇબ્રેરીઓ પ્રદાન કરીએ છીએ.

**ક્લાઈન્ટ લાઈબ્રેરીઓ**

મેસોસ્ફિયર બહુવિધ ભાષાઓમાં ઉપલબ્ધ છે. તમે તમારી મનપસંદ ભાષામાં તેનો ઉપયોગ કરી શકો છો. અમે હાલમાં Typescript, Python અને Rust ને સપોર્ટ કરીએ છીએ.

**આર્કિટેક્ચર**

મેસોસ્ફિયરને સરળ બનાવવા માટે બનાવવામાં આવ્યું છે, તમારા સર્વર કાર્યોને .../mesosphere ફોલ્ડરમાં લખો અને તેમને બેકએન્ડ પર જમાવો. અમારા [tutorials](.../demo/examples/js/tutorials/chat_app/) ને અનુસરીને Mesosphere સાથે પ્રયોગ કરવાનું શરૂ કરો. મેસોસ્ફિયર તમારા કાર્યો માટે ટાઈપ-સેફ API અને તમારો ડેટા સ્ટોર કરવા માટે પોસ્ટગ્રેસ ડેટાબેઝ બનાવવા માટે tRPC નો ઉપયોગ કરે છે.

## એકીકરણ

તમે અમારા [integration](.../demo/examples/python/integration/) નો ઉપયોગ કરીને LLM માં મેમરી ઉમેરી શકો તે માટે અમે [mem0](https://github.com/mem0ai/mem0) સાથે એકીકરણ સ્તર પ્રદાન કરીએ છીએ
