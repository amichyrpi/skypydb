## איך זה עובד

[Mesosphere](https://mesosphere.ahen-studio.com) הוא מסד נתונים יחסי והטמעות וקטוריות בקוד פתוח עם רישיון Apache 2.0, והוא נועד להיות פשוט, מהיר וקל לשימוש עבור מפתחי אתרים ומפתחי קצה. קרא, כתוב נתונים ובצע היגיון קשיח על ידי כתיבת פונקציות ב-Typescript עם בטיחות סוג מלא.

Mesosphere מספקת מסד נתונים, שבו אתה כותב את הפונקציות שלך ב-Typescript ומאחסן את הנתונים שלך במסד נתונים יחסי. אתה יכול גם לאחסן את הקבצים שלך במסד נתונים. אתה יכול גם ליצור, לחפש ולמחוק אוספי וקטורים כל זה בזמן אמת. אנו מספקים מספריות לקוח לאינטראקציה עם הפונקציות שכתבת.

**ספריות לקוחות**

Mesosphere זמין במספר שפות. אתה יכול להשתמש בו בשפה האהובה עליך. כרגע אנחנו תומכים ב-Typescript, Python ו-Rust.

**אַדְרִיכָלוּת**

Mesosphere עשוי להיות פשוט, כתוב לך פונקציות שרת בתיקייה ./mesosphere ופרוס אותן ל-backend. התחל להתנסות עם Mesosphere על ידי ביצוע [tutorials](./demo/examples/js/tutorials/chat_app/) שלנו. Mesosphere השתמש ב-tRPC כדי ליצור API בטוח מסוגים עבור הפונקציות שלך ומסד נתונים Postgres לאחסון הנתונים שלך.

## הִשׁתַלְבוּת
