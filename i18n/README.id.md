## Bagaimana cara kerjanya

[Mesosphere](https://mesosphere.ahen-studio.com) adalah database penyematan relasional dan vektor sumber terbuka dengan Lisensi Apache 2.0. Ini dirancang agar sederhana, cepat, dan mudah digunakan untuk pengembang web dan pengembang backend. Membaca, Menulis data, dan menjalankan logika keras dengan menulis fungsi dalam Naskah dengan keamanan tipe penuh.

Mesosphere menyediakan database, tempat Anda menulis fungsi Anda dalam TypeScript dan menyimpan data Anda dalam database relasional. Anda juga dapat menyimpan file Anda dalam database. Anda juga dapat membuat, mencari dan menghapus semua koleksi vektor secara realtime. kami menyediakan beberapa perpustakaan klien untuk berinteraksi dengan fungsi yang Anda tulis.

**Perpustakaan Klien**

Mesosfer tersedia dalam berbagai bahasa. Anda dapat menggunakannya dalam bahasa favorit Anda. Saat ini kami mendukung TypeScript, Python, dan Rust.

**Arsitektur**

Mesosphere dibuat sederhana, tuliskan fungsi server Anda di folder ./mesosphere dan terapkan ke backend. Mulailah bereksperimen dengan Mesosphere dengan mengikuti [tutorials](./demo/examples/js/tutorials/chat_app/) kami. Mesosphere menggunakan tRPC untuk membuat API yang aman untuk fungsi Anda dan database Postgres untuk menyimpan data Anda.
