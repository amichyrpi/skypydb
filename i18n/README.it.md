## Come funziona

[Mesosphere](https://www.usemesosphere.com/) è un database relazionale e di incorporamento vettoriale open source con licenza Apache 2.0, progettato per essere semplice, veloce e facile da usare per sviluppatori web e sviluppatori backend. Leggi, scrivi dati ed esegui logica complessa scrivendo funzioni in Typescript con piena sicurezza dei tipi.

Mesosphere fornisce un database in cui scrivi le tue funzioni in Typescript e memorizzi i tuoi dati in un database relazionale. Puoi anche archiviare i tuoi file in un database. Puoi anche creare, cercare ed eliminare raccolte di vettori, tutto questo in tempo reale. forniamo più librerie client per interagire con le funzioni che hai scritto.

**Biblioteche clienti**

Mesosphere è disponibile in più lingue. Puoi usarlo nella tua lingua preferita. Attualmente supportiamo Typescript, Python e Rust.

**Architettura**

Mesosphere è fatto per essere semplice, scrivi le funzioni del tuo server nella cartella .../mesosphere e distribuiscile nel backend. Inizia a sperimentare con Mesosphere seguendo il nostro [tutorials](.../demo/examples/js/tutorials/chat_app/). Mesosphere utilizza tRPC per creare un'API indipendente dai tipi per le tue funzioni e un database Postgres per archiviare i tuoi dati.

## Integrazione

Forniamo un livello di integrazione con [mem0](https://github.com/mem0ai/mem0) per aggiungere memoria a un LLM utilizzando il nostro [integration](.../demo/examples/python/integration/)
