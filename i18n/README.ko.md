## 작동 원리

[Mesosphere](https://www.usemesosphere.com/)은 Apache 2.0 라이센스가 있는 오픈 소스 관계형 및 벡터 임베딩 데이터베이스로, 웹 개발자와 백엔드 개발자가 간단하고 빠르며 쉽게 사용할 수 있도록 설계되었습니다. 완전한 유형 안전성을 갖춘 Typescript로 함수를 작성하여 데이터를 읽고 쓰고 하드 로직을 수행합니다.

Mesosphere는 Typescript로 함수를 작성하고 데이터를 관계형 데이터베이스에 저장하는 데이터베이스를 제공합니다. 파일을 데이터베이스에 저장할 수도 있습니다. 또한 벡터 컬렉션을 실시간으로 생성, 검색 및 삭제할 수도 있습니다. 우리는 귀하가 작성한 함수와 상호 작용할 수 있는 여러 클라이언트 라이브러리를 제공합니다.

**클라이언트 라이브러리**

Mesosphere는 여러 언어로 제공됩니다. 원하는 언어로 사용하실 수 있습니다. 현재 Typescript, Python 및 Rust를 지원합니다.

**건축학**

Mesosphere는 간단하게 만들어졌습니다. .../mesosphere 폴더에 서버 기능을 작성하고 백엔드에 배포합니다. [tutorials](.../demo/examples/js/tutorials/chat_app/)을 따라 Mesosphere 실험을 시작해 보세요. Mesosphere는 tRPC를 사용하여 함수에 대한 유형 안전 API를 생성하고 Postgres 데이터베이스를 사용하여 데이터를 저장합니다.

## 완성

우리는 [integration](.../demo/examples/python/integration/)을 사용하여 LLM에 메모리를 추가할 수 있도록 [mem0](https://github.com/mem0ai/mem0)과 통합 계층을 제공합니다.
