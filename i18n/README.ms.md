## Bagaimana ia berfungsi

[Mesosphere](https://www.usemesosphere.com/) ialah pangkalan data pembenaman perhubungan dan vektor sumber terbuka dengan Apache 2.0 Berlesen ia direka bentuk agar ringkas, pantas dan mudah digunakan untuk pembangun web dan pembangun bahagian belakang. Baca, Tulis data dan lakukan logik keras dengan menulis fungsi dalam Typescript dengan keselamatan jenis penuh.

Mesosphere menyediakan pangkalan data, tempat anda menulis fungsi anda dalam Typescript dan menyimpan data anda dalam pangkalan data hubungan. Anda juga boleh menyimpan fail anda dalam pangkalan data. Anda juga boleh membuat, mencari dan memadam koleksi vektor semua itu dalam masa nyata. kami menyediakan berbilang perpustakaan pelanggan untuk berinteraksi dengan fungsi yang anda tulis.

**Perpustakaan Pelanggan**

Mesosfera tersedia dalam pelbagai bahasa. Anda boleh menggunakannya dalam bahasa kegemaran anda. Kami kini menyokong Typescript, Python dan Rust.

**Seni bina**

Mesosfera dibuat untuk menjadi mudah, tulis fungsi pelayan anda dalam folder .../mesosphere dan gunakannya ke bahagian belakang. Mula bereksperimen dengan Mesosfera dengan mengikuti [tutorials](.../demo/examples/js/tutorials/chat_app/) kami. Mesosphere menggunakan tRPC untuk mencipta API jenis selamat untuk fungsi anda dan pangkalan data Postgres untuk menyimpan data anda.
