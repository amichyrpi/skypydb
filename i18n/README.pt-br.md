## Como funciona

[Mesosphere](https://mesosphere.ahen-studio.com) é um banco de dados relacional e de incorporação vetorial de código aberto com licença Apache 2.0 e foi projetado para ser simples, rápido e fácil de usar para desenvolvedores web e desenvolvedores back-end. Leia, escreva dados e execute lógica rígida escrevendo funções em Typescript com total segurança de tipo.

Mesosphere fornece um banco de dados, onde você escreve suas funções em Typescript e armazena seus dados em um banco de dados relacional. Você também pode armazenar seus arquivos em um banco de dados. Você também pode criar, pesquisar e excluir coleções de vetores em tempo real. fornecemos várias bibliotecas cliente para interagir com as funções que você escreveu.

**Bibliotecas de clientes**

Mesosfera está disponível em vários idiomas. Você pode usá-lo em seu idioma favorito. Atualmente oferecemos suporte a Typescript, Python e Rust.

**Arquitetura**

O Mesosphere foi feito para ser simples, escreva as funções do servidor na pasta ./mesosphere e implante-as no backend. Comece a experimentar o Mesosphere seguindo nosso [tutorials](./demo/examples/js/tutorials/chat_app/). Mesosphere usa tRPC para criar uma API de tipo seguro para suas funções e um banco de dados Postgres para armazenar seus dados.
