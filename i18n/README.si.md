## එය ක්රියා කරන ආකාරය

[Mesosphere](https://www.usemesosphere.com/) යනු Apache 2.0 බලපත්‍රයක් සහිත විවෘත-මූලාශ්‍ර සම්බන්ධක සහ දෛශික කාවැද්දීම් දත්ත සමුදායක් වන අතර එය සරල, වේගවත් සහ වෙබ් සංවර්ධකයින් සහ පසුපෙළ සංවර්ධකයින් සඳහා භාවිතා කිරීමට පහසු ලෙස නිර්මාණය කර ඇත. සම්පූර්ණ ආකාරයේ ආරක්ෂාවක් සහිතව Typescript හි කාර්යයන් ලිවීමෙන් දත්ත කියවීම, ලිවීම සහ දැඩි තර්කනය සිදු කිරීම.

Mesosphere දත්ත සමුදායක් සපයයි, එහිදී ඔබ ඔබේ කාර්යයන් Typescript වලින් ලියා ඔබේ දත්ත සම්බන්ධක දත්ත ගබඩාවක ගබඩා කරයි. ඔබට ඔබේ ගොනු දත්ත ගබඩාවක ගබඩා කළ හැකිය. ඔබට දෛශික එකතු කිරීම් තත්‍ය කාලීනව නිර්මාණය කිරීමට, සෙවීමට සහ මකා දැමීමටද හැකිය. ඔබ ලියූ කාර්යයන් සමඟ අන්තර් ක්‍රියා කිරීමට අපි බහු සේවාදායක පුස්තකාල සපයන්නෙමු.

**සේවාලාභී පුස්තකාල**

Mesosphere බහු භාෂා වලින් ලබා ගත හැක. ඔබට එය ඔබේ ප්රියතම භාෂාවෙන් භාවිතා කළ හැකිය. අපි දැනට Typescript, Python සහ Rust සඳහා සහය දක්වයි.

**ගෘහ නිර්මාණ ශිල්පය**

Mesosphere සරල ලෙස සාදා ඇත, .../mesosphere ෆෝල්ඩරය තුළ ඔබගේ සේවාදායක කාර්යයන් ලියා ඒවා පසු අන්තයට යොදවන්න. අපගේ [tutorials](.../demo/examples/js/tutorials/chat_app/) අනුගමනය කිරීමෙන් Mesosphere සමඟ අත්හදා බැලීම ආරම්භ කරන්න. Mesosphere ඔබගේ ක්‍රියාකාරකම් සඳහා TRPC භාවිතා කරමින් TRPC භාවිතා කරයි.
