## Funkcie

- Relačné: vytvorte svoje funkcie a uložte svoje údaje do relačnej databázy.

- Vkladanie vektorov: vytváranie, vyhľadávanie a odstraňovanie kolekcií vektorov.

- Ukladanie súborov: uložte svoje súbory do databázy.

- Pamäť: pridajte pamäť do LLM pomocou [mem0](https://github.com/mem0ai/mem0) a nášho [integration](./demo/integration/mem0/).

- Bezplatný a otvorený zdroj: Apache 2.0 s licenciou

- Viacplatformové: Windows, Linux, MacOS

## HttpClients

### TypeScript

Mesosphere ponúka klienta TypeScript na interakciu s databázou, môžete použiť buď vektorovú databázu vloženia alebo relačnej databázy. Najprv sa pozrieme na to, ako používať databázu vektorových vložení.

#### Vektorový model

Na vytvorenie vektorových vložení môžete použiť poskytovateľa modelov stromov AI.

- [x] HuggingFace Sentence Transformers
- [x] Ollama
- [x] OpenAI

Tu je príklad, ako používať tohto odloženého poskytovateľa.

```ts
import { httpClient } from "mesosphere";

// Sentence Transformers provider

// Create a client
async function use_sentence_transformers_provider(): Promise<void> {
  const client = httpClient({
    api_url: "http://localhost:8000",
    api_key: "local-dev-key",
    embedding_provider: "sentence-transformers",
    embedding_model_config: {
      model: "all-MiniLM-L6-v2",
    },
  });
}
```

```ts
import { httpClient } from "mesosphere";

// Ollama provider

// Create a client
async function use_ollama_provider(): Promise<void> {
  const client = httpClient({
    api_url: "http://localhost:8000",
    api_key: "local-dev-key",
    embedding_provider: "ollama",
    embedding_model_config: {
      model: "mxbai-embed-large",
      base_url: "http://localhost:11434",
    },
  });
}
```

```ts
import { httpClient } from "mesosphere";

// OpenAI provider

// Create a client
async function use_openai_provider(): Promise<void> {
  const client = httpClient({
    api_url: "http://localhost:8000",
    api_key: "local-dev-key",
    embedding_provider: "openai",
    embedding_model_config: {
      api_key: "your-openai-api-key",
      model: "text-embedding-3-small",
    },
  });
}
```

Po vytvorení klienta ho môžete použiť na interakciu s databázou.

Pridajte údaje do svojej vektorovej databázy.

```ts
  try {
    // Create a vector database or get it if it already exists
    const vectordb = await client.get_or_create_collection("my-videos");

    // Add data to your vector database
    await vectordb.add({
      data: ["Video Theo1", "Video Theo2"], // data to add
      metadatas: [{ source: "youtube" }, { source: "dailymotion" }], // metadata to add to the data
      ids: ["vid1", "vid2"], // unique ids for the data
    });
  } finally {
    // Close local resources
    await client.close();
  }
}
```

Odstráňte údaje z vašej vektorovej databázy.

```ts
  try {
    // Create a vector database or get it if it already exists
    const vectordb = await client.get_or_create_collection("my-videos");

    // Firts add data to your vector database
    // Add data to your vector database
    await vectordb.add({
      data: ["Video Theo1", "Video Theo2"], // data to add
      metadatas: [{ source: "youtube" }, { source: "dailymotion" }], // metadata to add to the data
      ids: ["vid1", "vid2"], // unique ids for the data
    });

    // Delete data from your vector database
    await vectordb.delete({
      by_ids: ["vid1", "vid2"],
      // by_metadatas: [{ source: "youtube" }, { source: "dailymotion" }] // delete by metadatas
      // by_data: ["Video Theo1", "Video Theo2"] // delete by data
    });
  } finally {
    // Close local resources
    await client.close();
  }
}
```

Dopytujte dáta z vašej vektorovej databázy.

```ts
  try {
    // Create a vector database or get it if it already exists
    const vectordb = await client.get_or_create_collection("my-videos");

    // Firts add data to your vector database
    // Add data to your vector database
    await vectordb.add({
      data: ["Video Theo1", "Video Theo2"], // data to add
      metadatas: [{ source: "youtube" }, { source: "dailymotion" }], // metadata to add to the data
      ids: ["vid1", "vid2"], // unique ids for the data
    });

    // Query for similar data
    const results = await vectordb.query({
      query_texts: ["This is a query"],
      number_of_results: 2,
    });

    // Access results
    for (let index = 0; index < results.ids[0].length; index += 1) {
      const doc_id = results.ids[0][index];
      const document = results.documents?.[0]?.[index];
      const distance = results.distances?.[0]?.[index];
      console.log(`${doc_id}, ${document}, ${distance}`);
    }
  } finally {
    // Close local resources
    await client.close();
  }
}
```

### Python

Mesosphere ponúka klienta Python na interakciu s databázou, môžete použiť buď vektorovú databázu vloženia alebo relačnej databázy. Najprv sa pozrieme na to, ako používať databázu vektorových vložení.

#### Vektorový model

Na vytvorenie vektorových vložení môžete použiť poskytovateľa modelov stromov AI.

- [x] HuggingFace Sentence Transformers
- [x] Ollama
- [x] OpenAI

Tu je príklad, ako používať tohto odloženého poskytovateľa.

```python
# Sentence Transformers provider
import mesosphere

# Create a client
client = mesosphere.HttpClient(
    api_url="http://localhost:8000",
    api_key="local-dev-key",
    embedding_provider="sentence-transformers",
    embedding_model_config={"model": "all-MiniLM-L6-v2"},
)
```

```python
# Ollama provider
import mesosphere

# Create a client
client = mesosphere.HttpClient(
    api_url="http://localhost:8000",
    api_key="local-dev-key",
    embedding_provider="ollama",
    embedding_model_config={
        "model": "mxbai-embed-large",
        "base_url": "http://localhost:11434",
    },
)
```

```python
# OpenAI provider
import mesosphere

# Create a client
client = mesosphere.HttpClient(
    api_url="http://localhost:8000",
    api_key="local-dev-key",
    embedding_provider="openai",
```

Po vytvorení klienta ho môžete použiť na interakciu s databázou.

Pridajte údaje do svojej vektorovej databázy.


Dopytujte dáta z vašej vektorovej databázy.

```python
# Create a vector database or get it if it already exists
vectordb = client.get_or_create_collection("my-videos")

# Firts add data to your vector database
# Add data to your vector database
vectordb.add(
    documents=["Video Theo1", "Video Theo2"],  # data to add
    metadatas=[
        {"source": "youtube"},
        {"source": "dailymotion"},
    ],  # metadata to add to the data
    ids=["vid1", "vid2"],  # unique ids for the data

results = vectordb.query(query_texts=["This is a query"], n_results=2)

# Access results
if not results:
```
