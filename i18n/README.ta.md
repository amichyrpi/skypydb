## இது எப்படி வேலை செய்கிறது

[Mesosphere](https://www.usemesosphere.com/) என்பது Apache 2.0 உரிமம் பெற்ற ஒரு திறந்த மூல தொடர்பு மற்றும் வெக்டர் உட்பொதித்தல் தரவுத்தளமாகும், இது வலை உருவாக்குநர்கள் மற்றும் பின்தள டெவலப்பர்கள் எளிதாகவும், வேகமாகவும், பயன்படுத்துவதற்கு எளிதாகவும் வடிவமைக்கப்பட்டுள்ளது. முழு வகைப் பாதுகாப்புடன் டைப்ஸ்கிரிப்டில் செயல்பாடுகளை எழுதுவதன் மூலம் தரவைப் படிக்கவும், எழுதவும் மற்றும் கடினமான தர்க்கத்தைச் செய்யவும்.

மீசோஸ்பியர் ஒரு தரவுத்தளத்தை வழங்குகிறது, அங்கு நீங்கள் உங்கள் செயல்பாடுகளை டைப்ஸ்கிரிப்டில் எழுதி, உங்கள் தரவை தொடர்புடைய தரவுத்தளத்தில் சேமிக்கிறீர்கள். உங்கள் கோப்புகளை தரவுத்தளத்திலும் சேமிக்கலாம். நீங்கள் நிகழ்நேரத்தில் வெக்டர் சேகரிப்புகளை உருவாக்கலாம், தேடலாம் மற்றும் நீக்கலாம். நீங்கள் எழுதிய செயல்பாடுகளுடன் தொடர்பு கொள்ள பல கிளையன்ட் லைப்ரரிகளை நாங்கள் வழங்குகிறோம்.

**வாடிக்கையாளர் நூலகங்கள்**

மீசோஸ்பியர் பல மொழிகளில் கிடைக்கிறது. உங்களுக்கு விருப்பமான மொழியில் இதைப் பயன்படுத்தலாம். நாங்கள் தற்போது Typescript, Python மற்றும் Rust ஆகியவற்றை ஆதரிக்கிறோம்.

**கட்டிடக்கலை**

மீசோஸ்பியர் எளிமையானதாக உருவாக்கப்பட்டுள்ளது, உங்கள் சேவையக செயல்பாடுகளை ../mesosphere கோப்புறையில் எழுதி பின்தளத்தில் பயன்படுத்தவும். எங்களின் [tutorials](../demo/examples/js/tutorials/chat_app/) ஐப் பின்தொடர்வதன் மூலம் மீசோஸ்பியருடன் பரிசோதனையைத் தொடங்கவும். உங்கள் செயல்பாடுகளுக்கு ஒரு வகை-பாதுகாப்பான API ஐ உருவாக்க மெசோஸ்பியர் tRPC ஐப் பயன்படுத்துகிறது மற்றும் உங்கள் தரவைச் சேமிக்க Postgres தரவுத்தளத்தை உருவாக்குகிறது.

## ஒருங்கிணைப்பு

எங்களின் [integration](../demo/examples/python/integration/) ஐப் பயன்படுத்தி LLMக்கு நினைவகத்தைச் சேர்க்க [mem0](https://github.com/mem0ai/mem0) உடன் ஒரு ஒருங்கிணைப்பு அடுக்கை வழங்குகிறோம்.
