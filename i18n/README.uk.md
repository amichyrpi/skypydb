## Як це працює

[Mesosphere](https://www.usemesosphere.com/) — це реляційна та векторна база даних із відкритим вихідним кодом із ліцензією Apache 2.0, її розроблено, щоб бути простим, швидким і легким у використанні для веб-розробників і розробників серверної частини. Читайте, записуйте дані та виконуйте жорстку логіку, записуючи функції в Typescript із повною безпекою типів.

Mesosphere надає базу даних, де ви пишете свої функції на Typescript і зберігаєте свої дані в реляційній базі даних. Ви також можете зберігати свої файли в базі даних. Ви також можете створювати, шукати та видаляти колекції векторів у реальному часі. ми надаємо декілька клієнтських бібліотек для взаємодії з функціями, які ви написали.

**Клієнтські бібліотеки**

Мезосфера доступна кількома мовами. Ви можете використовувати його своєю улюбленою мовою. Зараз ми підтримуємо Typescript, Python і Rust.

**Архітектура**

Мезосфера створена так, щоб бути простою, запишіть свої серверні функції в папку ../mesosphere та розгорніть їх у серверній частині. Почніть експериментувати з Mesosphere, дотримуючись наших [tutorials](../demo/examples/js/tutorials/chat_app/). Mesosphere використовує tRPC для створення надійного API для ваших функцій і бази даних Postgres для зберігання ваших даних.
